# SocraCode 🚀

### AI-powered LeetCode learning assistant

While solving LeetCode problems, many students get stuck. Sometimes they don't understand the error in their approach, and sometimes they struggle to understand the problem itself.

AI assistants can help, but many of them directly provide the complete solution. This removes the thinking process that helps developers build problem-solving skills.

SocraCode was built to solve this problem. It acts as an AI mentor inside LeetCode by providing **progressive hints, guidance, complexity analysis, and test cases** instead of immediately revealing solutions.

------------------------------------------------------------------------

## ✨ Features

-    **Progressive Hints**\
    Get guidance that gradually pushes you toward the solution without spoiling the answer.

-    **Complexity Analysis**\
    Understand the efficiency of your approach and learn possible optimizations.

-    **Test Case Generation**\
    Generate additional test cases, including edge cases, to debug your solution.

-    **Automatic Problem Detection**\
    Extracts the current LeetCode problem and provides context-aware AI assistance.

------------------------------------------------------------------------

## ⚙️ How it works

    LeetCode Problem
        ↓
    Content Script extracts problem context
        ↓
    Extension sends context to LLM
        ↓
    AI generates structured hints and analysis
        ↓
    SocraCode displays guidance inside LeetCode

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   JavaScript
-   HTML/CSS
-   Chrome Extension APIs (Manifest V3)
-   LLM API integration

------------------------------------------------------------------------

## 🚀 Installation

1.  Clone the repository

``` bash
git clone https://github.com/deepikavudari/SocraCode.git
```

2. Open Chrome extensions:

```
chrome://extensions/
```

3.  Enable **Developer Mode**

4.  Click **Load unpacked** and select the project folder.

5.  Open LeetCode and start solving with SocraCode.

------------------------------------------------------------------------

SocraCode aims to make AI a learning companion rather than a shortcut, helping developers improve their DSA intuition and become better problem solvers.
